# L'atelier de Lilwenn - Portfolio

Portfolio interactif présentant mes projets, mon parcours et mes compétences sous forme d'un atelier virtuel.

## Installation

1. Clonez ou téléchargez ce dépôt
2. Ouvrez `index.html` dans votre navigateur
3. Aucune dépendance externe requise !

## Structure du projet

- `index.html` - Page principale
- `styles/` - Feuilles de style CSS
  - `main.css` - Styles de base
  - `atelier.css` - Styles de l'espace atelier
  - `potager.css` - Styles de l'espace potager
  - `modal.css` - Styles des modales
- `scripts/` - Fichiers JavaScript
  - `content.js` - Contenu des modales
  - `navigation.js` - Navigation entre les espaces
  - `modal.js` - Gestion des modales
- `assets/` - Ressources multimédias
  - `images/` - Images du portfolio
  - `files/` - Fichiers téléchargeables (CV)

## Fonctionnalités

- Navigation entre l'atelier (projets techniques) et le potager (soft skills)
- Objets interactifs avec modales d'information
- Design responsive
- Animations fluides

## Personnalisation

1. Modifiez le contenu dans `scripts/content.js`
2. Ajoutez vos images dans `assets/images/`
3. Remplacez le CV par le vôtre dans `assets/files/`

## Compatibilité

Compatible avec tous les navigateurs modernes (Chrome, Firefox, Safari, Edge)
